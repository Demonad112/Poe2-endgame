poe.ninja proxy

Server-side proxy for the PoE2 Endgame Companion's character-import feature. It exists only because poe.ninja's public character API can't be called directly from a browser (CORS + an SSE first hop). Everything it returns is public data already visible on a character's poe.ninja profile.

Endpoint: /api/character?account=<acct>&league=<slug>&character=<name>
